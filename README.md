# Job Search Agent

A multi-agent job application automation system built with CrewAI, FastAPI, and Streamlit.

## Features
- Search real government jobs from USAJobs API
- AI-powered job analysis using Gemini
- Automatic resume customization
- Cover letter generation
- LinkedIn outreach message drafting

## Tech Stack
Python · CrewAI · FastAPI · Streamlit · Gemini API · USAJobs API
